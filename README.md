#### Crie um novo projeto Nest.js
```powershell
nest new nome_do_projeto
```

#### Inicie o servidor 
```powershell
nest start run
```
Ou
```powershell
npm start:dev run
```

#### Criar um recurso completo
```powershell
nest g resource
```
